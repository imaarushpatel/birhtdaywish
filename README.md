# Khushi Birthday Surprise Website

## New Flow
1. Open `index.html` (mystery intro page)
2. Click `Tap To Reveal`
3. It opens `surprise.html` with no-scroll animated scenes

## Media Already Connected
Photos and video from `assets/photos` and `assets/videos` are already mapped in `script.js`.

## Optional Customization
- Main message: `config.message` in `script.js`
- Photo captions: `config.photoCaptions` in `script.js`
